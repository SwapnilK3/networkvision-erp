üöÄ How to Run the Prototype

Option 1: Upgrade Node.js (Recommended)

  1. Install Node.js 14+ from nodejs.org
  2. Navigate to project directory: cd /home/swapnil/Avishkar/networkvision-erp
  3. Install dependencies: npm install
  4. Start development server: npm run dev
  5. Open browser to: http://localhost:3000

Option 2: Use Docker (Alternative)

  1. Install Docker on your system
  2. Create a Dockerfile with Node.js 18+ base image
  3. Build and run the container
  4. Access the application via container port mapping

Option 3: Static Build

  1. Use a system with Node.js 14+ to build: npm run build
  2. Serve the dist folder using any web server
  3. Example: python -m http.server 3000 (from dist folder)

Option 4: Cloud Development Environment

  1. Use GitHub Codespaces, Gitpod, or CodeSandbox
  2. Clone the repository in the cloud environment
  3. Run the development server in the cloud
  4. Access via the provided cloud URL
